About this app

Instructions for using the app

  • The initial interface is five-button with four-way navigation.
  • This 5-key interface allows the users to control the navigation go up, left, down, and right through controlling 'w', 'a', 's' and 'd.
  • Press 'Enter' to select number or operation.

Description of test

  • Using my finger and elbow to complete both tests.
  • During the test, entering the equation '12 / 56 ='.
  • Computing the average time of five trials for both finger and elbow.

Description of result

  • It takes 7.21 seconds for my fingers to complete the test.
  • And it takes 36.47 seconds for my elbow to complete the test.
  • It's hard for the elbow to do the input as the keyboard is relatively small. Finger could adapt to the 5-key interface easily.
